# Nuka Market Fanmade

A fanmade Fallout-inspired marketplace built with React, TypeScript and Node.js.

## Features

* Product shop
* Dynamic filters
* Product categories
* Sorting system
* Responsive UI
* Fallout inspired design

## Tech Stack

Frontend:

* React
* TypeScript
* Vite

Backend:

* Node.js
* Prisma

## Installation

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
npm install
npm run dev
```
